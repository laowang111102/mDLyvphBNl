# PHP＋MySQL个人博客文章管理系统 java941

## 项目概述

本项目是一款基于PHP和MySQL的个人博客文章管理系统，包含前台用户模块和后台管理员模块，为用户提供了一个方便、高效的管理与发布文章的平台。

## 技术栈

- 后端：PHP
- 数据库：MySQL
- 集成环境：phpstudy、Wampserver、Xampp

## 功能模块

### 前台用户模块

- 用户注册与登录
- 浏览管理员发布的个人介绍
- 浏览作品信息和日常信息
- 查看文章详情
- 发布留言给管理员

### 后台管理员模块

- 管理作品信息（增加、删除、修改、查询）
- 管理留言内容和前台用户
- 修改首页图片和个人介绍

## 系统角色

- 前台用户：负责浏览和评论文章
- 后台管理员：负责管理和发布文章，处理留言

## 运行环境

- PHP版本：5.6至8之间

## 部署步骤

1. 确保已安装合适版本的PHP和MySQL环境（建议使用集成环境）
2. 导入数据库文件
3. 将代码部署至服务器
4. 配置服务器和数据库连接
5. 访问前台与后台进行相应操作

## 目录结构

```
/
├── admin/               # 后台目录
├── css/                # 样式文件
├── images/             # 图片资源
├── js/                 # JavaScript文件

├── index.php           # 前台首页
├── admin.php           # 后台首页
└── database.sql        # 数据库文件
```
